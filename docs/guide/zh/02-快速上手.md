# 快速上手

本页用最少的命令跑通一个知识库:**初始化 → 投喂 → 提问 → 维护**。命令细节见 [CLI 命令](03-cli-命令.md)。

> 前置:已 `pip install guanlan-wiki`(见 [安装](01-安装.md))。`ingest` / `query` 需配置模型;其余命令零 LLM、可离线。

## 1. 初始化一个知识库

```bash
guanlan init my-wiki     # 在新目录初始化
guanlan init             # 或就地初始化当前目录
```

`init` 是**确定性的(零 LLM)**,已存在的文件**不会被覆盖**,可安全重复运行。

生成结构:

```
my-wiki/
├── AGENTAO.md       # Agent 行为约束 + 指针
├── SCHEMA.md        # 本库 Schema:领域 / 启用页面类型 / 自定义规则
├── raw/             # 原始资料(只读,事实来源)
└── wiki/            # Agent 全权生成的知识层
    ├── index.md     # 全量页面目录
    ├── log.md       # append-only 时间线
    └── overview.md  # 跨资料活体综述
```

> **`-C` 指定知识库根**:除 `init` 外,其余命令都用 `guanlan -C my-wiki <命令>` 指向知识库目录(也可先 `cd my-wiki` 再省略 `-C`)。

## 2. 投喂资料

把一篇 markdown 资料喂给 Agent,它会读 `raw/` 里的源、生成/更新 `wiki/` 里的页面:

```bash
guanlan -C my-wiki ingest path/to/source.md
```

> 非 `.md` 文件(PDF / DOCX / PPTX / …)先用 [`guanlan convert`](07-多格式转换.md) 转成 `raw/<slug>.md`,再 `ingest`。
>
> **`raw/` 只读不可变**:Agent 只读原始资料、永不修改,写门禁会在每次写操作前后对 `raw/` 做快照校验。

## 3. 提问

```bash
guanlan -C my-wiki query "你的问题"
```

默认是只读问答(基于已建好的 wiki)。若想把这次综合**沉淀回 wiki**(一次受门禁约束的写入),用回填:

```bash
guanlan -C my-wiki query "你的问题" --backfill
```

## 4. 维护(零 LLM、可离线)

```bash
guanlan -C my-wiki check     # frontmatter / 断链 / 来源校验
guanlan -C my-wiki health    # stub 页面 + index↔disk 同步(--strict → 退出码 6)
guanlan -C my-wiki lint      # 孤儿页 / 断链 / 缺失实体
guanlan -C my-wiki graph     # 写出 graph/graph.json + graph.html(--json-only 跳过 html)
```

`health` / `lint` 是 **advisory**(建议性,不阻断);`check` 是确定性校验。各命令的退出码语义见 [维护:体检与图谱](04-维护-体检-图谱.md)。

## 5. 检索

```bash
guanlan -C my-wiki search "关键词"     # 零 LLM 确定性 BM25 + CJK 全页召回
```

## 可选:在浏览器里用

```bash
pip install 'guanlan-wiki[web]'
guanlan -C my-wiki web                 # 起本地 Web 宿主,仅监听 127.0.0.1,默认开浏览器
guanlan -C my-wiki web --port 9000 --no-browser
```

浏览器里可:浏览 wiki 并跟随 `[[wikilink]]` 导航、跑 check·health·lint、看 graph、从 `raw/` 触发 ingest、与 agent 只读多轮对话。**仅供本机单用户——绝不要把端口暴露到网络。** 详见 [Web 宿主](05-web-宿主.md)。

## 可选:暴露给 MCP 客户端

```bash
pip install 'guanlan-wiki[mcp]'
guanlan -C my-wiki mcp                 # stdio 上起只读 MCP 服务端
```

详见 [MCP 宿主](06-mcp-宿主.md)。
